Чек листы

Чек-лист — инструмент контроля качества, включающий список обязательных проверок для тестируемого ресурса. Он обеспечивает прозрачность, полноту и управляемость тестирования.

Тест кейсы

Тест-кейс представляет собой структурированный набор шагов с указанием входных данных, условий и ожидаемого результата, предназначенный для верификации работы конкретной функции приложения.

  • Тест кейсы

Баг репорты

Баг-репорт представляет собой структурированное описание обнаруженного дефекта, включающее данные о его проявлении, уровне серьёзности, приоритете и окружении, в котором возникла ошибка.
